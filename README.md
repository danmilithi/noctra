# 🎬 Noctra

Plataforma de streaming inspirada na Netflix, com sistema de perfis, catálogo dinâmico e preview de trailers.

## 🚀 Funcionalidades

- Seleção de perfis
- Catálogo de filmes e séries
- Carrossel dinâmico
- Preview de trailers ao passar o mouse

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
